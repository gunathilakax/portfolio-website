My Projects

Bookmark Buddy

2025

A feature-rich web browser extension designed to simplify bookmark management and enhance browsing productivity. Bookmark Buddy offers a responsive search bar, sorting options, customizable view modes, dark/light mode toggling, and full-screen functionality. Built with JavaScript, HTML, CSS, and the Chrome Bookmarks API, this project highlights my expertise in web development and UI/UX design.

Hutch CliQ UI Redesign

2023

A redesign of the Hutch mobile app to enhance usability and modernize its design. Key improvements include enhanced usage visibility, simplified balance management, streamlined add-ons, and a unified navigation bar. The clean and modern layout ensures a seamless user experience while maintaining Hutch brand identity.

Lab and Lecture Hall Reservation System

2024

A web-based system designed to streamline the reservation of labs and lecture halls in Sri Lankan government universities. Lecturers can check availability and reserve resources online, reducing manual errors and optimizing resource utilization. This system enhances academic workflow and supports efficient management of university facilities.

Traffic Sign Recognition System

2024

An Arduino-based system designed to recognize traffic signs, including speed limits, and dynamically adjust vehicle speed accordingly. The system also provides voice instructions to the driver, explaining the meaning of each detected sign. By integrating image processing with Arduino, this project enhances road safety and driver awareness.

EduPara

2023

EduPara is an innovative online platform connecting educators, students, and institutes. Teachers showcase their expertise through targeted ads, while students explore diverse profiles and classes. Institutes promote programs to a targeted audience, enhancing visibility and enrollment. Designed to foster a dynamic learning community.

Facial Expression Recognition System

2022

A machine learning-based system designed to interpret human emotions through facial expression recognition. By analyzing facial features like eyes, eyebrows, nose, and mouth, the system identifies emotions such as happiness, sadness, and anger. This project leverages advanced algorithms and data preprocessing techniques to bridge the gap between human emotions and machine understanding.
